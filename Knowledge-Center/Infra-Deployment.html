Work Activities 

1. Creation of Subscription and create master service principle (atomai-deployment-master-sp)for                       automatic deployment of full stack.

 

Creating a service principal

You can create a new or use an existing service principal in your Azure Active Directory tenant.

App registration

  • Navigate to the Azure portal.
  • Select Azure Active Directory from the left-hand side menu
  • Select App registrations and + New registration
  • Enter a name for the application (the service principal name).
  • Select Accounts in this organizational directory only.
  • For Redirect URI select Web and enter any URL you want; it doesn't have to be real or work.
  • Then select Register.

Create a subscription

Use the following procedure to create a subscription for yourself or for someone in the current Azure Active Directory. When you're done, the new subscription is created immediately.

  • Sign in to the Azure portal.
  • Navigate to Subscriptions and then select Add.
  • On the Create a subscription page, on the Basics tab, type a Subscription name.
  • Select the Billing account where the new subscription will get created.
  • Select the Billing profile where the subscription will get created.
  • Select the Invoice section where the subscription will get created.
  • Next to Plan, select Microsoft Azure Plan for DevTest if the subscription will be used for development or testing workloads. Otherwise, select Microsoft Azure Plan.
  • Select the Advanced tab.
  • Select your Subscription directory. It's the Azure Active Directory (Azure AD) where the new subscription will get created.
  • Select a Management group. It's the Azure AD management group that the new subscription is associated with. You can only select management groups in the current directory.
  • Select one or more Subscription owners. You can select only users or service principals in the selected subscription directory. You can't select guest directory users. If you select a service principal, enter its App ID.
  • Select the Tags tab.
  • Enter tag pairs for Name and Value.
  • Select Review + create. You should see a message stating Validation passed.
  • Verify that the subscription information is correct, then select Create. You'll see a notification that the subscription is getting created.

After the new subscription is created, the owner of the subscription can see it in the Subscriptions page.
