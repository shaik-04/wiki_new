This button will take you to the Feature Store of Atom.Ai where you can check all the feature platform for ML that Atom.Ai provides.       

 Navigation – Atom.AI ⇢ Feature Store

Main Screen (user action)

Different Collection of Schemas

     Navigation – Atom.AI

      Link – Click here

The Main thing in the homepage of Atom.Ai is the different group of schemas from where you can pick any schema that you want in your project by just clicking on any one of the four groups present (UX Works, Solution Works, Data Works and Algo Works).

      Navigation – Atom.AI  ⇢ solution 

       Link – Click here

For Example, if you click on the Data Works, so the next window will contain Data Works as the first group header followed by other groups. It follows the hierarchy of UX Works ⇢ Data Works ⇢ Algo Works ⇢ Solution Works. This hierarchy works as a circular list and after Solution Works, it again starts with UX Works.
