How to create Solutions

Atom.AI

USER ACTION REQUIRED

There are two ways to create a solution in Atom.Ai

  • Without adding anything in the cart
  1. Open homepage of Atom.Ai
  2. Goto “My Workspace” from the left side panel
  3. Click on “My Solutions” Tab to create new solutions
  4. Create new solution by adding the product details and submit it
  • After adding schemas in the cart
  1. Select the desired schemas to the cart
  2. Clone the schemas
  3. Create new solution by adding the product details and submit it
  4. Open homepage of Atom.Ai
  5. Select any one collection of db schemas

 

Without adding anything in the cart (user action)

  • Opening the homepage of Atom.Ai and going to “My Workspace”

      Navigation – Atom.AI

      Link – Click here

      Navigation – Atom.AI ⇢ My Workspace

      Link – click here

Click on My Solutions tab to create new solution

      Navigation – Atom.AI ⇢ My Workspace ⇢ My Solutions

      Link – click here

Click on “+” symbol to create new solution.

  • Create new solution

      Navigation – Atom.AI ⇢ My Workspace ⇢ My Solution ⇢ Create New Solution

      Link – click here

Fill up the Solution details (to create new solution) : user action required.

  1. Fill the desired solution name
  2. Fill the desired solution short name
  3. Select the industry from the drop-down menu
  4. Select the solution type from the drop-down menu
  5. Write other specific detail you want to add regarding the solution
  6. Write a description about the solution
  7. Select solution start date and end date from the calendar
  8. Fill the budget of the solution

After filling all the details, click on submit button to create new solution.

 

After adding schemas in the cart (user action)

  • Opening the homepage of Atom.Ai and selecting the schemas

      Navigation – Atom.AI

      Link – Click here

Click on any one of the schemas group whose schemas you wanted to add to the cart.

      Navigation – Atom.AI ⇢ solution 

      Link – Click here

Select the desired schema to the cart by clicking on the encircled basket option.

  • Cloning the schemas

      Navigation – Atom.AI ⇢ solution

      Link – Click here

Click on the cart button to re-check the selected schemas.

If you want to delete the unwanted added schemas from the cart, click on the delete icon next to the schemas and click on ok button.

Click on clone button to clone the selected schemas from the cart.

      Navigation – Atom.AI ⇢ solution ⇢ provision-overview

      Link – click here

New window to select the solution will pop-up. To create new solution, click on Create New Product tab.

  • Create new solution

      Navigation – Atom.AI ⇢ solution ⇢ provision-overview ⇢ new-solution

      Link – click here

Fill up the Solution details (to create new solution) : user action required.

  1. Fill the desired solution name
  2. Fill the desired solution short name
  3. Select the industry from the drop-down menu
  4. Select the solution type from the drop-down menu
  5. Write other specific detail you want to add regarding the solution
  6. Write a description about the solution
  7. Select solution start date and end date from the calendar
  8. Fill the budget of the solution

After filling all the details, click on submit button to create new solution.

 

Verifying the Solution is created

Once, the solution is created, It will redirect to the ‘My Solutions’ page.

Navigation – Atom.AI ⇢ solution

Link – click here

You will be able to find your newly created solution here. Click on the newly created solution.

Here, you will be able to see the details of your solution. One thing to note here, During solution creation, if you choose solution type as POD  3 then deployment date won't be there under Development status tab.
