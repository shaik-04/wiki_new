Data Model

The data model demonstrates the relationships between the tables used by SCCT.

 

Overview of the Data Model

An overview into the dimensions and facts tables in SCCT Data Model:

 

Address

Primary Key field name: address_id

Foreign Key field name: customer_id

 

Foreign Key reference: tbl_customer_master.customer_id

Customer

Primary Key field name: customer_id

Order

Primary Key field name: order_id

Foreign Key field name: customer_id

 

Foreign Key reference: tbl_customer_master.customer_id

Shipment

Primary Key field name: ship_id

Foreign Key field name: customer_id

Foreign Key field name: order_id

Foreign Key field name: load_id

 

Foreign Key reference: tbl_customer_master.customer_id

Foreign Key reference: tbl_order_data.order_id

Foreign Key reference: tbl_load_data.load_id

Load

Primary Key field name: load_id

Foreign Key field name: carrier_scac

 

Foreign Key reference: tbl_carrier_master.carrier_scac

Carrier

Primary Key field name: customer_id
